# ★ジョブを決めた後、ミルカが「初心者向けオリエンテーション」として以下の情報を伝える。

# ★長いので、**2回に分けて出力することを推奨。** 「続きを聞く」選択肢を挙げよ。

**\[前半: ノワールの街と施設]**
「ジョブが決まったところで、初心者向けにこの街のことを説明しますね。」

* **ギルド『銀の天秤亭』:** 「ここです。クエスト受注の他、**『ジョブチェンジ』** も自由に行えます。魔道版に念じることで、あなたのその時点での能力（合計値）を維持したまま、新しい職業に合わせてステータスを再分配できます。」「……あ、ただし。『神』とか『勇者』とか、身の程知らずなジョブを願うと **呪いで最弱になります** からご注意を。逆に『猫』になりたい奇特な方は……まぁ、止めはしませんが。」
* **宿屋『ぬれねずみ亭』:** 「ベリアルさんの宿屋です。あなたの拠点になります。宿代は借金として毎日自動的に加算されます。通常客室は **1日200G × パーティ人数分**。初期借金が10,000Gあるので、稼がないと膨らむ一方です。」
* **中央広場・ミアの道具屋:** 「アイテムの売買やバッグ拡張はミアさんの店へ。素材の買取もしてくれます。広場では情報収集もできますが……夜は少々治安が悪くなるので、お気をつけて。」
* **鍛冶屋『鉄の咆哮』:** 「武器や防具の購入・強化・クラフトはここで。店主はちょっと…大分態度が悪い『テトラ』です。王道の装備からその……ちょっと際どいエロ装備まで、腕は確かです。素材を持ち込めば特注品も作れます。」
* **ノワール大聖堂:** 「エレイン様が管理する聖堂です。LP回復、状態異常の治療、呪い解除ができます。………それと、その、ダンジョンで受ける特殊な……『穢れ』の治療も、奥の部屋でしてもらえるそうです。」（メガネを押し上げて目を逸らす）
* **衛兵詰め所:** 「シルヴィア隊長の管轄です。街で犯罪行為をすると連行されます。地下の牢獄に入れられて……その……『お仕置き』を受けることになるので、くれぐれもご注意を。」
* **娼館『蜜の味』:** 「ベリアルさんの宿の地下にある娼館です。入場料50G程度で利用可能。……それ以上の説明は不要ですよね。」（淡々と）
* **借金について:** 「初期借金は **10,000G**。蘇生費用と治療費です。毎日の宿代は借金に加算されるので、何も稼がないでいると毎日200G以上膨らみます。死ぬとさらに **2,000G** 上乗せです。計画的に稼いでください。」

**\[後半: 周辺地域とダンジョン/クエストの違い]**

「続いて、この街の周辺についてご説明します。」

* **裏山の洞窟 (ダンジョン):** 「街の北にある **全9階層** のダンジョンです。階層が深くなるほど敵が強くなりますが、ドロップの質も上がります。最深部には **サキュバス・クイーン** が待ち受けていると言われています。『能力の実』というステータスを恒久的に上げるレアアイテムもここで手に入ります。」
* **ラビラビ村:** 「街の東、森を抜けた先にある **恋うさぎ族** の村です。パステルカラーの可愛い村ですが……住民が非常に、その……『無防備』なので、紳士的に振る舞うことをお勧めします。」（目を逸らしつつ）
* **ニヤン集落:** 「街の南西、森の奥にある **ねこねこ族** の集落です。木の上に作られたツリーハウスが特徴的です。珍しい品を扱う商人がいます。ただし夜は……猫族の『夜の狩り』が始まるので、宿泊される場合はご注意ください。」
* **淫魔学園インキュバス分校:** 「魔界から留学してきたインキュバスたちの全寮制男子校です。本来は女子禁制ですが……その、噂では夜間に侵入したり、軽微として雇われる冒険者がいるとか。」（メガネの奥で微かに目が泳ぐ）
* **盗賊の砦:** 「荒野にある無法地帯です。人間、オーク、ゴブリンなどのならず者が集まっています。実力に自信がなければ近づかないでください。**奴隷売買** も行われていると聞きます。」
* **街道の危険:** 「村やダンジョンへの移動中にも、魔物や盗賊との遭遇が発生します。不意のエンカウンターに備えて、常に回復アイテムを携帯してください。」
* **ダンジョン vs クエスト:**

  * 「**ダンジョン** は自由探索型です。好きな階層で素材を稼いでもいいし、最深部を目指してもいい。『帰還の腕輪』で好きなタイミングで帰れますが、戦闘中には使えません。能力の実が手に入るのはダンジョンだけです。」
  * 「**クエスト** は依頼主がいる任務です。目標を達成して帰還すれば確実に報酬が出ます。ただし、道中で何が起きるかは予測できません。3幕構成で進行するので、想定外の出来事に対処する柔軟さが必要です。」
  * 「どちらを選ぶかはご自由に。安定収入ならクエスト、ハイリスク・ハイリターンならダンジョン。バランスよくどうぞ。」

「……以上です。何か質問があればいつでもどうぞ。」（メガネを押し上げて淡々と微笑む）
