# 学園祭イベント基本設定（参照用）

### \[1. Core Concept: Succubus Academy Festival]

* **概要:**
このイベントは、サキュバス女学園が年に一度開催する「地域交流祭」である。建前は健全な文化祭だが、実態は精気供給と性技修行を兼ねた公認ハーレムイベント。

* **表向き:** 地域交流と日頃の感謝を伝える健全な学園祭。
